(static) getPolyfit(x, y, start, boxsize, nfit) → {object}

This function is based on lfit() and gaussj() from Numerical Recipes in C (Press, Teukolsky, Vetterling, and Flannery; Cambridge U. Press, 1992). It fits a polynomial of degree (nfit-1) to a set of boxsize points given by x[1...boxsize,2] and y[1...boxsize]. The return value is the sum of the squared errors (chisq) between the (x,y) pairs and the fitted polynomial and the fitness coefficients.
Parameters:
Name Type Description
x object Array containing the values of X.
y object Array containing the values of Y.
start object Start point in array containing the values of Y.
boxsize number The box size.
nfit number Order of detrending (2: linear, 3: quadratic, etc).

(static) getRScale(minbox, maxbox, boxratio, alg) → {object}

Allocates and fills rs[], the array of box sizes used by getDfa() below. The box sizes range from (exactly) minbox to (approximately) maxbox, and are arranged in a geometric series such that the ratio between consecutive box sizes is (approximately) boxratio (Peng algorithm). The return value is box sizes array.
Parameters:
Name Type Description
minbox number The minimum box size.
maxbox number The maximum box size.
boxratio number The ratio between successive box sizes.
alg number The algorithm for scaling.
